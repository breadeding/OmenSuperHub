# OmenSuperHub
OmenSuperHub is a lightweight alternative to HP's Omen Gaming Hub (OGH), offering powerful system control features without requiring an internet connection or including ads and bloat.

## ✨ Features
- Fan speed control and custom fan curves  
- CPU & GPU power limit adjustment  
- DB version switching  
- Custom Omen key actions  
- Real-time temperature and power monitoring  
- Silent operation from the system tray  
- Spoofs OGH presence without using it

## ✅ Supported Models
- Shadow Elf 8P, 8PP, 9, 9P, 10  
- Pavilion Gaming 9, 10  

## ❌ Unsupported Models
- Shadow Elf 6

> **Warning:** Running on unsupported models may cause missing sensor data, system instability, or blue screens.

## 🔧 Usage Tips
- **Close or uninstall OGH** before launching OmenSuperHub to avoid conflicts.
- Use the tray icon for quick profile switching and real-time data without locking power limits.
- For long-term use, **disable OGH auto-start** and enable auto-start for OmenSuperHub.
